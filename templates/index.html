Welcome to the TCG Helper API!

TCG Helper is a free API which helps consolidate current meta decks and an average price for the card decks including links.

Click the Barboach below to access documentation.

Version Information

Current version for the TCG Helper API is on Version 1.0 and is able to:

  • Produce Current Standard Meta Decks
  • Provide user with Cards associated with each decks
  • User can search, find all or random choice through API endpoints

Future version will calculate prices of each card and display them on JSON data.
